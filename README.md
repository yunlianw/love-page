# LovePage 恋爱单页系统

一个甜蜜的情侣纪念网站系统，记录你们的点点滴滴。

## 功能特色

- 💕 首页计时器（精确到秒）
- ✈️ 旅行足迹（地图标注）
- 📸 甜蜜相册
- 🎨 爱好展示
- 💝 一起做的事
- 📅 纪念日倒计时
- 🎵 背景音乐播放器
- 🔒 页面加密保护
- ⚡ 一键生成静态页面

## 安装要求

- PHP >= 7.4
- MySQL 5.7+ / MariaDB 10.3+
- PDO扩展
- MBString扩展

## 安装步骤

1. 上传安装包到网站目录
2. 解压：`tar -xzf love-page-v1.0.1.tar.gz`
3. 重命名目录（可选）：`mv love-page yoursite`
4. 访问网站：`https://your-domain.com/install/`
5. 按向导完成安装
6. 删除 `install/` 目录

## 目录结构

```
love-page/
├── config/          配置文件
├── core/            核心类库
├── install/         安装程序（安装后删除）
├── templates/       模板文件
├── assets/          静态资源
│   ├── avatars/     头像上传目录
│   ├── css/         样式文件
│   ├── js/          脚本文件
│   └── music/       音乐文件
└── admin/           后台管理（安装时可自定义目录名）
```

## 后台功能

- 基本设置（标题、日期、头像、Emoji）
- 栏目开关（控制各板块显示）
- 旅行足迹管理
- 甜蜜相册管理
- 爱好管理
- 一起做的事管理
- 纪念日管理
- 背景音乐管理
- 一键生成静态页面
- 密码修改、后台目录修改

## 技术栈

- 后端：PHP 8.x（兼容 7.4-8.2）
- 数据库：MySQL
- 前端：原生 HTML/CSS/JS
- 地图：Leaflet.js
- 播放器：原生 Audio API
